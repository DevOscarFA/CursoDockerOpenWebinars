# Curso de Docker
## Que es Docker
* Virtualización a niveñ de S.O
* Aislamiento de recurso a nivel de Kernel (cgroups, namespaces)
* Flexibilidad y portabilidad
* Enfocado a sistemas altemente distribuidos
